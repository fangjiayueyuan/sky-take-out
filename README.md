在application-dev.yml中配置本地数据库地址：

```
sky:
  datasource:
    driver-class-name: com.mysql.cj.jdbc.Driver
    host: localhost
    port: 3306
    database: sky_take_out
    username: root
    password: root123&&
```

mac启动nginx服务：

```
brew install nginx // 安装
brew services start nginx // 启动
brew services stop nginx // 停止
```
